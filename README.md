## 📌 Project Overview

This project presents a complete **India Air Pollution Analytics Dashboard** built using **Power BI** to analyze AQI (Air Quality Index) trends, pollutant distribution, and pollution severity across Indian states and cities.

The dashboard transforms raw air pollution data into meaningful and interactive visual insights to help understand:

* Air Quality Index (AQI) trends
* State-wise pollution levels
* Most & least polluted cities
* Pollutant contribution analysis
* Pollution severity categories
* Air quality visualization through maps and KPIs

**Dataset Coverage:** 70+ Cities | 17 States | Multiple Pollutants

---

## 🎯 Project Objectives

✔ Analyze overall AQI trends across India

✔ Visualize pollutant contribution (PM2.5, PM10, NO2, etc.)

✔ Build interactive and professional dashboard visuals
